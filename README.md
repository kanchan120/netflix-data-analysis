# netflix-data-analysis
"A comprehensive data analysis project on the Netflix dataset to clean data, handle missing values, and visualize trends like content distribution and top-rated categories."

"Movies vs TV Shows Comparison" 
A count plot showing the total number of Movies compared to TV Shows in the Netflix dataset. This analysis highlights that Movies make up the majority of the content available on the platform.

"Content Distribution by Country" 
A bar chart representing the top 10 countries on Netflix based on the volume of movies and TV shows produced. It clearly identifies the major content producers for the platform.
